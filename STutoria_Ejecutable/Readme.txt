- Para ejecutar el programa se necesita Visual Studio, en caso de no tener IDE se adjunta un archivo ejecutable STutoria.App.application 
- Primero se debe hacer la instalación del programa, para lo cual se debe ejecutar el archivo setup.exe
- Modificar el documento STutoria.App.exe.config.deploy (con un editor de texto) que se encuentra en la dirección
  STutoria_Ejecutable\Application Files\STutoria.App_1_0_0_1\
  Se debe modificar la cadena de conexión para que funcione en un equipo diferente al que se usó para publicar el proyecto
- Ahora ya puede ejecutar el programa STutoria.App.application
